# نظام إدارة الطلبات

نظام بسيط لإدارة طلبات الواتساب والانستقرام — يسجّل الطلبات، يطبع الفواتير، ويتابع الحالات.

## المميزات
- ✅ لوحة تحكم بإحصائيات يومية
- ✅ طلبات بعنوان كويتي كامل (منطقة، قطعة، شارع، جادة، بيت)
- ✅ منتجات جاهزة (تختار منها عند إنشاء الطلب)
- ✅ 4 حالات: جديد، قيد التجهيز، تم التوصيل، ملغي
- ✅ فاتورة قابلة للطباعة
- ✅ روابط مباشرة للاتصال والواتساب
- ✅ نسخ احتياطي (تصدير/استيراد)
- ✅ يشتغل على الجوال والكمبيوتر

## كيفية النشر على GitHub Pages

### الخطوة 1: إنشاء مستودع (Repository)
1. سجّل دخول على [github.com](https://github.com)
2. اضغط على الزر الأخضر **New** (أو علامة + فوق على اليمين → New repository)
3. اسم المستودع: أي اسم — مثلاً `orders` أو `my-shop`
4. خليه **Public** (لازم يكون عام عشان GitHub Pages يشتغل مجاناً)
5. فعّل خيار **Add a README file**
6. اضغط **Create repository**

### الخطوة 2: رفع ملف index.html
1. داخل المستودع، اضغط **Add file** → **Upload files**
2. اسحب ملف `index.html` أو اختره من جهازك
3. اكتب أي رسالة في صندوق "Commit changes" (مثلاً: "initial upload")
4. اضغط **Commit changes**

### الخطوة 3: تفعيل GitHub Pages
1. من داخل المستودع، روح لتبويب **Settings** (فوق باليمين)
2. من القائمة الجانبية، اضغط **Pages**
3. تحت **Source** اختر **Deploy from a branch**
4. **Branch**: اختر `main`، والمجلد `/ (root)`
5. اضغط **Save**

### الخطوة 4: افتح موقعك
- بعد دقيقة أو دقيقتين، فوق في نفس صفحة Pages راح يظهر رابط على شكل:
  ```
  https://USERNAME.github.io/REPO-NAME/
  ```
- هذا رابط موقعك! احفظه في المفضلة على جوالك.

## ملاحظات مهمة

### حفظ البيانات
- البيانات تتحفظ **على جهازك فقط** (في المتصفح)
- ما تنتقل تلقائياً بين الأجهزة
- كل فترة صدّر نسخة احتياطية من تبويب **الإعدادات** — خصوصاً قبل ما تغيّر جهاز أو تمسح بيانات المتصفح

### استخدام نفس البيانات على جوال وكمبيوتر
1. على الجهاز الأول: الإعدادات → **تصدير نسخة احتياطية**
2. أرسل الملف لنفسك (واتساب / إيميل)
3. على الجهاز الثاني: افتح الرابط → الإعدادات → **استيراد من ملف**

### تحديث الموقع لاحقاً
إذا تبي تعدّل على الملف:
1. افتح المستودع على GitHub
2. اضغط على `index.html`
3. اضغط على قلم التعديل (فوق يمين)
4. عدّل واضغط **Commit changes**
5. خلال دقيقة، الموقع يتحدّث تلقائياً
